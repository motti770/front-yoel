## 📝 זרימות עבודה מרכזיות לבדיקה

### זרימה 1: ליד → לקוח → הזמנה → משימות
```
1. יצירת ליד חדש ✓
2. קידום ליד בשלבים (Pipeline) ✓
3. המרת ליד ללקוח (כפתור "המר ללקוח") ✓
4. יצירת הזמנה ללקוח ✓
5. בחירת מוצרים והתאמה אישית ✓
6. יצירת משימות אוטומטית מההזמנה ✓
7. עובדים מקבלים ומבצעים משימות ✓
8. סיום הזמנה ✓
```

### זרימה 2: הגדרת מוצר חדש
```
1. יצירת תהליך עבודה (Workflow) עם שלבים ✓
2. יצירת פרמטרים (צבעים, מידות) ✓
3. יצירת המוצר ✓
4. שיוך תהליך העבודה למוצר ✓
5. שיוך פרמטרים למוצר ✓
6. המוצר מוכן להזמנות ✓
```
