# 🎭 Mock Mode - מדריך שימוש

**תאריך:** 29 בדצמבר 2025

---

## 🚀 מה זה Mock Mode?

Mock Mode הוא מצב עבודה שמאפשר לך לראות את **כל האפליקציה עובדת** ללא Backend!

כל הנתונים מזויפים (Mock) אבל **נראים אמיתיים לחלוטין** 💯

---

## ⚡ איך להפעיל/לכבות?

**קובץ:** `src/services/api.js`

### להפעיל Mock Mode:
```javascript
const MOCK_MODE = true;  // 👈 שנה ל-true
```

### לכבות Mock Mode (כשהבק אנד יעבוד):
```javascript
const MOCK_MODE = false;  // 👈 שנה ל-false
```

**זהו! רק שינוי אחד** ✅

## 🎯 איך זה עובד?

כל פונקציה ב-services בודקת אם `MOCK_MODE = true`:

```javascript
getAll: async () => {
    if (MOCK_MODE) {
        // החזר נתונים מזויפים
        return { success: true, data: mockLeads };
    }
    // אחרת - קרא לבק אנד אמיתי
    return api.get('/leads');
}
```
