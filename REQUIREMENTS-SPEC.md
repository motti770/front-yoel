## 🎯 מטרת המערכת

מערכת CRM (ניהול לקוחות) מלאה לניהול מפעל יודאיקה:
- ניהול לידים ומכירות
- ניהול לקוחות

## 🎣 עיקרון מרכזי: "לא דג - אלא חכה"

### ⚠️ חשוב מאוד להבין:

המערכת צריכה להיות **גמישה וניתנת להתאמה** על ידי המנהל בעצמו.

**פעם ראשונה:** אנחנו מקימים ומזינים נתונים ראשוניים.
**מכאן והלאה:** המנהל עצמאי - יכול להוסיף מוצרים, לשנות תהליכים, להתאים.

### מה המנהל שולט בו בעצמו (ללא מפתח):

| אזור | מה מגדיר | בעצמו? |
|------|----------|--------|
| **מוצרים** | סוגי מוצרים חדשים, פרטים, תמונות | ✅ כן |
| **פרמטרים** | אפשרויות התאמה לכל מוצר (צבעים, מידות...) | ✅ כן |
| **תהליכי עבודה** | שלבים לכל מוצר, סדר, זמנים | ✅ כן |
| **שיוך שלבים** | מי אחראי - מחלקה פנימית / ספק חיצוני | ✅ כן |
| **ספקים** | הוספת ספקי חומרים/שירותים חדשים | ✅ כן |
| **התראות** | מתי לשלוח ולמי (לכל שלב) | ✅ כן |
| **הודעות** | תוכן ההודעות ללקוחות | ✅ כן |
| **תשלומים** | שיטת תשלום למוצר (מקדמה/מלא) | ✅ כן |
| **משלוח** | אפשרויות משלוח | ✅ כן |
| **מחלקות** | הוספת מחלקות ועובדים | ✅ כן |
| **קבוצות** | ארגון לקוחות/לידים בקבוצות | ✅ כן |

### דוגמה - איך זה עובד:

```
היום: המנהל רוצה להוסיף מוצר חדש "כיסוי ספר תורה"

1. נכנס לניהול מוצרים → הוסף מוצר
2. מגדיר פרמטרים: צבע עור, סוג כיתוב, גודל
3. יוצר תהליך עבודה חדש:
   שלב 1: עיצוב (מעצבת סקיצה)
   שלב 2: חריטה (ספק חיצוני - שם הספק)
   שלב 3: הרכבה (עובדי ייצור)
   שלב 4: בקרת איכות (מנהל)
4. מגדיר התראות:
   - אחרי שלב 2 → שלח ללקוח "המוצר בחריטה"
   - אחרי שלב 4 → שלח ללקוח "מוכן לאיסוף"
5. שומר - ומהיום הזמנות של המוצר הזה עוברות את התהליך הזה!
```
